# Health-Ease Landing Page

A modern, responsive landing page for a healthcare booking platform that allows users to skip clinic queues and manage appointments digitally.

## 🚀 Features

* Appointment booking section
* Clear "How It Works" process
* Testimonials for credibility
* Call-to-action for conversions
* Responsive layout (mobile-friendly)

## 🛠️ Built With

* HTML5
* CSS3 (Flexbox & Grid)
